# Jogo da Cobrinha

## Descrição
Projeto desenvolvido para praticar lógica de programação e manipulação do JavaScript.

## Funcionalidades
- Interações dinâmicas na página
- Atualização de conteúdo via JavaScript

## Tecnologias Utilizadas
- HTML5
- CSS3
- JavaScript

## Objetivo
Fortalecer a base em JavaScript e lógica aplicada.

## Status
Projeto de estudo.
